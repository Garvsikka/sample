# LevelUp Academy - Complete MERN Stack

Delhi's First & Only Institute to Develop Games for Maths using AI!

## 🚀 Quick Start

### Prerequisites
- Node.js (v16+)
- MongoDB (Local or Atlas)
- npm or yarn

### Installation

```bash
# Navigate to project
cd levelup-academy-website

# Install all dependencies
npm run install-all

# Setup environment variables
cp frontend/.env.example frontend/.env
cp backend/.env.example backend/.env

# Update .env files with your configuration

# Start development servers
npm run dev
